Welcome to the Multi-Agent AI Workshop. In this step, you'll configure your environment.

## Prerequisites

- Gemini API Key from Google AI Studio
- Optional: Firebolt and Gmail credentials (sandbox is enabled by default)

## Environment Variables

Create a `.env.local` file with the following:

<CodeBlock language="bash" code={`GEMINI_API_KEY=your_gemini_api_key_here\nFIREBOLT_CLIENT_ID=workshop_sandbox_xyz\nFIREBOLT_CLIENT_SECRET=your_firebolt_secret\nFIREBOLT_ACCOUNT=workshop_account\nFIREBOLT_DATABASE=workshop_db\nNEXT_PUBLIC_APP_URL=http://localhost:3000\nNODE_ENV=development`} />

## Verify

Run the dev server and open the homepage.

<CodeBlock language="bash" code={`npm install\nnpm run dev`} />

If the demo loads, you're ready for Step 2.

<Checkpoint items={["Homepage loads without errors","/workshop route renders","Env variables created in .env.local"]} />
